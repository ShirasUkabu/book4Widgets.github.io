How to ask questions#

（如何提问）#

There is a “raise hand” button but instead, please use the chat or the Q/A. Prefer the chat, we will be able to answer your questions and interact in real-time through the chat. If need be we can have the host stop presenting to answer a question that’s especially interesting or pertinent. We’re not able to answer the Q/A questions unless we stop the flow of the tutorial to answer the question. （有一个“举手”按钮，但请使用聊天或问答。更喜欢聊天，我们将能够回答您的问题并通过聊天实时互动。如果需要，我们可以让主持人停止演示，回答特别有趣或相关的问题。除非我们停止教程的流程来回答问题，否则我们无法回答 Q/A 问题。）

Jupyter lab essentials#

（Jupyter 实验室基础知识）#

  • To open a notebook: double-click it in the file browser. （要打开notebook：在文件浏览器中双击它。）

  • To hide the file browser: click on the folder icon. （要隐藏文件浏览器：单击文件夹图标。）

  • To run a single code cell in the notebook: Push Shift and Enter, or use the “play” button. （要在notebook中运行单个代码单元：按 Shift 和 Enter，或使用“播放”按钮。）

  • To add a new view for an output, right-click on an output and select “Create new view for output”. （要为输出添加新视图，请右键单击输出并选择“Create new view for output”（为输出创建新视图）。）

  • Use tab to autocomplete and shift-tab to show documentation. （使用 tab 自动完成，使用 shift-tab 显示文档。）

  • Use CTRL + “/” to comment/uncomment code（使用 Ctrl + “/” 注释/取消注释代码）

Interactive Jupyter widgets#

(交互式 Jupyter 小组件)#

A Python widget is an object that represents a control on the front end, like a slider. A single control can be displayed multiple times - they all represent the same python object. (Python 小组件是表示前端控件的对象，类似于滑块。单个控件可以多次显示 - 它们都表示同一个 python 对象。)

import ipywidgets as widgets

slider = widgets.FloatSlider(
    value=7.5,
    min=5.0,
    max=10.0,
    step=0.1,
    description='Input:',
)

slider

You can trigger actions in the kernel when a control value changes by “observing” the value. Here we set a global variable when the slider value changes. （当控制值发生变化时，您可以通过 “观察” 该值来触发内核中的操作。在这里，当滑块值发生变化时，我们设置一个全局变量。）

square = slider.value * slider.value

def handle_change(change):
    global square
    square = change.new * change.new
    
slider.observe(handle_change, 'value')

square

56.25

interact: a shortcut for examining functions#

（interact：检查函数的快捷方式）#

The interact function generates widgets based on a function’s arguments and displays any output from the function. （interact 函数根据函数的参数生成小组件，并显示该函数的任何输出。）

def f(x):
    print(x * x)

f(9)

81

import ipywidgets as ipw

widgets.interact(f, x=(0, 100));

Behind the scenes#

（幕后故事）#

Behind the scenes, even pure Python widgets are composed of two pieces: （在幕后，即使是纯 Python 小部件也由两部分组成：）

  • Python, which runs in the notebook kernel. +（Python，在 notebook 内核中运行。）

  • JavaScript, which runs in the browser. +（JavaScript，它在浏览器中运行。）

When writing your own widgets, that means making a choice: write only in Python or write in both Python and Javascript. （在编写自己的小部件时，这意味着要做出选择：只用 Python 编写或同时使用 Python 和 Javascript 编写。）

Which to choose depends on what you are trying to accomplish. This tutorial will focus on writing your own widgets in pure Python. An example of a pure-Python package that includes some interesting interfaces is reducer, a package for calibrating astronomical data. （选择哪个取决于您要完成的任务。本教程将重点介绍如何用纯 Python 编写你自己的小部件。包含一些有趣接口的纯 Python 包的一个示例是 reducer，这是一个用于校准天文数据的包。）

Jupyter widgets as a framework#

（Jupyter 小部件作为框架）#

Jupyter widgets forms a framework for representing python objects interactively. Some large open-source interactive controls based on Jupyter widgets include: （Jupyter 小部件构成了一个以交互方式表示 python 对象的框架。一些基于 Jupyter 小组件的大型开源交互式控件包括：）

  • bqplot - 2d plotting library in which everything displayed is a widget （bqplot - 2D 绘图库，其中显示的所有内容都是小部件）

  • ipympl - widget backend for matplotlib graphics （IPYMPL - matplotlib 图形的小部件后端）

  • pythreejs - low-level 3d graphics library （PyThreeJS - 低级 3D 图形库）

  • ipyvolume - 3d plotting and volume rendering （ipyvolume - 3D 绘图和体积渲染）

  • ipyleaflet_ - interactive maps （ipyleaflet_ - 交互式地图）

  • ipywebrtc - video streaming （IPYWebRTC - 视频流）

  • ipysheet - interactive spreadsheets （IPYSHEET - 交互式电子表格）

  • ipytree - tree for viewing hierarchical material （ipytree - 用于查看分层材料的树）

  • ipycanvas - interactive drawing in a notebook （ipycanvas - 笔记本中的交互式绘图）

  • ipyevents - capture mouse/keyboard events on a widget （ipyevents - 捕获 Widget 上的鼠标/键盘事件）

  • …
